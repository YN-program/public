## 「国民に血を流す覚悟を」──玉木雄一郎の発言

### 発言の内容

国民民主党の玉木雄一郎は、憲法改正の議論において次のように述べました:

> 「国民に血を流す覚悟を」

{{<x user="sh56300" id="1851893180637356246">}}
&nbsp;

### この発言の何が問題なのか

**1. 国民を「手段」として扱う思想**

民主主義社会では:
- 国家は国民のために存在します
- 国民は国家の「手段」ではありません

しかし、「国民に血を流す覚悟を」という発言は:
```
国家のために国民が血を流す
    ↓
国民は国家の手段
    ↓
国家が目的、国民が手段
```

これは、民主主義の根本的な転倒です。

**2. 「血を流す」という言葉の意味**

「血を流す」とは、つまり:
- 戦争で死ぬこと
- 命を捧げること

政治家が国民に「死ぬ覚悟」を求める──これは、どのような国家を目指しているのでしょうか?

**3. 憲法改正との関連**

なぜ、憲法改正の文脈でこの発言が出るのでしょうか?

答えは明白です:
```
憲法改正
    ↓
自衛隊の国軍化
    ↓
海外派兵の容易化
    ↓
国民の徴兵・動員
    ↓
「血を流す」
```

**「血を流す覚悟」とは、戦争ができる国への転換を前提とした発言です。**

### 誰が「血を流す」のか

重要な問いがあります:

**稲田氏自身は、血を流すのでしょうか?**

歴史を見れば、答えは明白です:
```
【戦前】
政治家・軍上層部: 安全な場所で指揮
国民・若者: 戦場で「血を流す」

【これから?】
政治家: 安全な場所で「国を護れ」と命令
国民・若者: 戦場へ
```

**「血を流せ」と命じる人は、自分では血を流しません。**

### 「国を護る」の欺瞞

「国を護る」とは、何を護ることなのでしょうか?

- 国民の命と尊厳を護ること?
- それとも、「国家体制」を護ること?

もし本当に「国民」を護るなら:
```
国民の命を守る
    ↓
戦争を避ける
    ↓
外交努力、平和構築
```

しかし、「血を流せ」と言うとき:
```
「国家」「体制」を守る
    ↓
そのために国民の命を使う
    ↓
国民は「手段」
```

**ここでも、国家と国民の転倒が起きています。**

## 「ナチスに学べ」──麻生太郎元首相の発言

### 発言の内容

麻生太郎は、憲法改正に関連して次のように述べました:

> 「ナチスの手口に学んだらどうか」
> 「誰も気づかないうちに、あの手口に学んで、ワイマール憲法が変わっていた」

### この発言の深刻さ

**1. ナチスの「手口」とは何か**

ナチスがワイマール憲法を事実上無効化した「手口」:

```
【1933年2月】国会議事堂放火事件
    ↓
【1933年2月28日】緊急大統領令
「国民と国家を保護するための大統領令」
→ 基本的人権を停止
    ↓
【1933年3月23日】全権委任法
→ 議会を経ずに法律制定可能
→ 憲法に反する法律も制定可能
    ↓
【結果】民主主義の終焉
```

**これが「ナチスの手口」です。**

**2. 「誰も気づかないうちに」という部分**

麻生太郎は何を言っているのでしょうか?

```
「誰も気づかないうちに」憲法を変える
    ↓
国民が気づかないように
    ↓
静かに、合法的に
    ↓
民主主義を破壊する
```

**これは、「ナチスに学んで、国民に気づかれないように憲法を変えよう」という提案です。**

**3. 後の釈明の欺瞞**

麻生太郎は後に自らの発言を撤回しました。

しかし、発言を見れば明白です:
- 「学んだらどうか」= 提案
- 「あの手口に学んで」= 肯定的
- 「誰も気づかないうちに」= 望ましいこととして

**これは批判ではなく、賞賛です。**

### 現在進行中の「ナチスの手口」

では、日本で「ナチスの手口」は使われているでしょうか?

```
【ナチス】
緊急大統領令 → 基本的人権停止
    ↓
全権委任法 → 議会の無力化

【日本?】

    ↓
```
